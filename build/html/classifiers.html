Submodules¶

classifiers.classifier module¶

classifiers.classifier.classify(data: DataFrame)¶

Performs state classification using I1Q1,I2Q2 data.

Returns labels 0: ground 1: excited 2: f

classifiers.classifier.population(arr: array)¶

Returns populations for each state (0, 1, 2).

classifiers.classifier.probabilities(arr: array)¶

Returns probabilities for each state (0, 1, 2) as P_g, P_e, and P_f respectively.

classifiers.classifier.reshape_for_exp(data: Series, reps: int, num_steps: int)¶

This takes in the list of I or Q data and rearranges it such that we have num steps columns and reps rows.
